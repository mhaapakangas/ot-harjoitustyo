## Tetris-peli
Tetris on geometrinen ongelmanratkaisupeli, jossa pelaaja asettelee erimuotoisia, putovia palikoita ruudukossa.
Tavoitteena on kerätä pisteitä muodostamalla palikoista täysiä rivejä.
